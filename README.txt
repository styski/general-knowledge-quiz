GENERAL KNOWLEDGE QUIZ APP – JAK TO DOSTAT NA WEB

1) Stáhni tento ZIP a rozbal ho do počítače.

2) Otevři GitHub a svůj repository.

3) Pokud je repository prázdný:
   - klikni na "Upload files"
   - přetáhni do okna VŠECHNY soubory z rozbalené složky quiz-app
   - klikni dole na "Commit changes"

4) Otevři Vercel.

5) Klikni na "Add New" -> "Project".

6) Vyber svůj GitHub repository s nahranými soubory.

7) Vercel si většinou vše pozná automaticky.
   Pokud se zeptá:
   - Framework: Vite
   - Build Command: npm run build
   - Output Directory: dist

8) Klikni na "Deploy".

9) Po chvíli dostaneš veřejnou adresu webu.

POKUD BYS CHTĚL WEB UPRAVIT:
- otevři v GitHubu soubor src/App.jsx
- klikni na ikonu tužky
- uprav texty nebo otázky
- klikni na Commit changes
- Vercel změnu sám znovu nasadí

POKUD CHCEŠ SPUSTIT WEB NA SVÉM POČÍTAČI:
- musíš mít nainstalovaný Node.js
- v terminálu ve složce projektu spusť:
  npm install
  npm run dev
